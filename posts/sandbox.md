## これは何か

ブログを Markdown で書き始めることにした。これはそのテストだ。まずは画像でも表示してみよう。

{{# image }}
  theme/title_logo.png
{{/ image }}

このブログは
[ruhoh](http://ruhoh.com/) を用いて生成している。
標準の Markdown に画像の文法は無いので、小さなプラグインを作って、以下の形式で書けるようにした。

    // 実際には波括弧 2 つで書く
    {# image }
       path/to/image.png
    {/ image }

と、このようにソースコードの表示もぬかりない。
